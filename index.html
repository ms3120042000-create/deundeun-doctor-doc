🩺 든든의사

방문진료 의사를 위한 AI 기반 진료 보조 시스템 — 리포트 자동 생성부터 일정 관리, 진료기록부 작성까지

🚀 서비스 바로가기 GitHub
💡 서비스 개요 01
방문진료 의사는 환자 집에서 진료 후 보호자에게 구두로 설명하고 끝나는 경우가 많습니다.
든든의사는 체크리스트 입력 하나로 AI가 보호자용 리포트를 자동 생성하고, 방문 일정과 진료기록까지 한 곳에서 관리합니다.
🤖
AI 리포트 자동 생성
혈압·식욕·통증·컨디션 입력 시 보호자 전달용 자연어 리포트를 GPT가 즉시 작성
📅
방문 일정 캘린더
환자별 방문 예약을 달력으로 관리. 드래그로 날짜 변경, 상세정보 툴팁 제공
📋
진료기록부 작성
활력징후·진단·처방을 입력하면 의료법 양식의 진료기록부를 실시간 미리보기 및 PDF 출력
💾
임시저장 & 자동저장
로컬스토리지 기반 30초 자동저장. 새로고침해도 작성 내용이 유지됨
🔄 사용 흐름 02
단계행동결과
1방문 전 — 캘린더에 환자 예약 등록주소·연락처·날짜 저장, 달력에 표시
2방문 중 — 체크리스트 입력 (혈압·식욕·통증·기분)—
3AI 리포트 생성 버튼 클릭보호자 전달용 리포트 자동 작성
4리포트 복사 → 보호자에게 전달카카오톡 등으로 즉시 공유 가능
5진료기록부 작성 (선택)의료 양식 PDF 출력 가능
